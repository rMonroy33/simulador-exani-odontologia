# Simulador EXANI-II Odontología UMSNH

Plataforma web estática para practicar ingreso a Odontología con formato de tres opciones por reactivo.

## Funciones principales

- Modo práctica rápida.
- Modo simulacro completo.
- Modo repasar errores.
- Preguntas agrupadas por área.
- Todas las preguntas del área aparecen en una sola pantalla.
- Lecturas largas en comprensión lectora.
- Casos aplicados en matemáticas, biología y ciencias de la salud.
- Ilustraciones SVG integradas en algunos reactivos.
- Inglés diagnóstico, separado del puntaje calificable.
- Resultados por área.
- Temas débiles sugeridos.
- Revisión de respuestas y explicaciones al final.
- Guardado local de preguntas falladas para repaso posterior.

## Archivos

```txt
index.html
style.css
script.js
data/questions.json
data/questions_extra.json
data/questions_enhanced.json
```

## Publicación

Compatible con GitHub Pages. Los archivos deben estar en la raíz del repositorio y GitHub Pages debe apuntar a `main` y `/root`.

## Nota

Este simulador es una herramienta de práctica no oficial. Las preguntas deben revisarse y ampliarse continuamente con base en los temarios oficiales vigentes.
